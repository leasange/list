## Description 

List is a library to handle with slice in golang easily, it's inspired by the Class List .NET 

https://msdn.microsoft.com/pt-br/library/6sh2ey19(v=vs.110).aspx

## Installation

```bash
go get github.com/lucasvmiguel/list
```
